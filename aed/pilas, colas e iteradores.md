 # pilas
estructura de datos con comportamiento "último en entrar, primero en salir" (LIFO). siempre sale primero el de encima, el que entró al último. es como una pila de libros
## operaciones típicas
- push -> añade elementos a la pila
- pop -> elimina el último elemento ingresado
- isempty -> está vacía la pila ¿?
- isfull -> está llena la pila ¿?
- top -> retorna el elemento que ingresó al último
# colas
estructura de datos con comportamiento "primero en entrar, primero en salir" (FIFO). siempre sale primero el de abajo, el que entró primero. es como una cola de la tienda

## operaciones típicas
- enqueue -> añade elementos al final de la cola
- dequeue -> elimina el primer elemento ingresado
- isempty -> está vacía la cola ¿?
- isfull -> está llena la cola ¿?
- front -> retorna el elemento que está primero en la cola
