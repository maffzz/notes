- servidor -> hardware con más potencia de almacenamiento que los cpus o ram o discos
- gabinetes (racks) -> "armario" donde se almacenan los servidores y los provee de energía
- data center -> lugar en donde se almacenan muchos gabinetes con muchos servidores
- economía de escala -> mientras hayan más data centers hay más gabinetes y más servidores, por lo que el costo por servidor disminuye. las empresa utilizan esto para alquilar servidores o máquinas virtuales
- on-premises -> implementar tu propio data center en tus instalaciones
- core -> un sistema que cuando se cae afecta muchísimo al cliente | ejemplo: caída del sistema de un banco

beneficios del cloud computing:
- agilidad -> toma muy poco tiempo crear una máquina virtual
- bajo costo -> solo se cobra por el tiempo que se utiliza, por lo que puedes eliminar la máquina virtual cuando te sea favorable
- autogestión por el usuario

proveedores líderes:
1. AWS (2006)
2. Microsoft Azure (2008)
3. Google Cloud (2008)

¿qué es el cloud computing o computación en la nube?
es la entrega bajo demanda de capacidad informática para desplegar servicios en la nube.
