### conceptos fundamentales
- linux fue creado en 1991, basado en unix de los 70's
- usuarios -> entidad (usuario o aplicación) que otorga permisos que permitan a los demás usuarios o aplicaciones realizar tareas específicas
	- creados por instalación
	- usuario root -> usuario principal. administra el sistema operativo
- grupos -> es obligatorio que un usuario pertenezca a 1 grupo primario y a 0 o n grupos adicionales
- en linux existe un grupo que se llama **sudo : super user do**. cualquier usuario que pertenezca a ese grupo va a poder utilizar comandos de administrador como si fuera el root
